Guia de HTML

Estructura: <html>

La estructura del documento html es simple:
Para empezar declararemos el tipo de documento con: <!DOCTYPE html> ,despues abriremos la etiqueta html y subdividiremos el documento en 3 partes:

HEAD:
Es la cabecera del documento. En ella se puede declarar titulos, estilos, enlaces, y la codificación.
BODY:
Es el cuerpo del documento. Es lo que se visualiza en el navegador.
FOOT:
Es el pié del documento.

Head: <head>

En la cabecera del documento se encuentran:

TITLE: <title>
Es el título que cogerá el navegado para presentarlo en la parte superior.
META: <meta>
Es el tipo de codificación que usará el navegador para visualizar el documento.
LANG: <lang>
Es el lenguaje en el que está escrito del documento.
STYLE: <style>
Es donde introduciremos los estilos de CSS para dar formato a la pagina.

Body:<body>

En el cuerpo del documento se encuentran:

Elementos de bloque:
Pueden contener otros elementos de línea o de bloque y siempre comienzan en una linea nueva
Elementos de Linea:
Solo contienen datos y a otros elementos de línea.

Ahora listaremos algunos elementos de bloque y linea e indagaremos en ellos:

Manejo de texto: <h1...h6>
Establecen encabezados y diferentes tamaños de títulos. Es un elemento de línea. Se clasifican de h1 (el mas grande) a h6 (el mas pequeño.)

Manejo de texto: <p>
Establece una agrupacion de lineas de texto en un parrafo. En el texto los saltos de linea deben ser explicitos, para ello utilizaresmos la etiqueta <br> Tambien podemos hacer que se visualize una linea horizontal que sirve para separar parrafos o temas entre si. Otra etiqueta que podemos encontrar es: <wbr>, la cual sirve para crear saltos de linea en una palabra si el navegador reduce su formato.

Hay otras etiquetas en el manejo de texto que permite establecer los diferentes tipos de fuentes. Todas ellas se definen dentro del parrafo o etiqueta <p>.

La etiqueta <b> establece el texto en negrita.
La estiqueta <i> establece el texto en cursiva.
La etiquta <u> subraya todo el texto
La etiqueta <s> o <strike> tacha el texto.
La etiqueta <big> establece un tamaño superior al texto.
L etiqueta <small> establace un tamaño de texto inferior

Por ultimo hay un tipo de texto el cual permite que el navegador lo visualice tal como aparece en el documento respetando tabulaciones, saltos de linea y blancos. Se representa con la etiqueta <pre>
Manejo de listas: Lista de elementos no ordenada: <ul>

Manejo de listas: Lista de elementos ordenados: <ol>
Las listas de elementos ordenados se componen también de dos etiquetas: <ol> que es la que indica que es una lista ordenada y <li> que indica un elemento de la lista. En este tipo de lista podemos destacar dos atributos: <type> que especifica el tipo de simbolo que utiliza, <start> que indica en que numero o simbolo debe empezar a contar la lista y <reversed> que lista los numeros al reves.

Manejo de listas: Listas de definiciones: <dl>
Las listas de definiciones se definen por tres etiquetas: <dl> que es la que indica que es una lista de definiciones, <dt> que especifica el termino a definir y <dd> que es la definicion de este termino.

Enlaces: <a>
Los enlaces en html sirven para establecer el origen y el destino de un hipervinculo. Con el atributo href indicamos a que destino nos llevará el enlace.
Por ejemplo, os dejo un enlace a la página de lorem ipsum. Este enlace contiente su dirección, pero puedo renombrarla, en vez de poner su url. Además de la url tambien se puede acceder a otros tipos de direcciones.
Imagenes: <img>
Esta etiqueta de linea sirve para introducir una imagen gracias al atributo src y a la direccion del archivo.Con los atributos width y height delimitamos el tamaño a visualizar la imagen y con alt le damos un pequeño titulo en caso de que la imagen no se visualizase. Tablas: <table>
Esta etiqueta inserta una tabla de tamaño variable que puede contener filas, columnas, filas de encabezado,filas de pié, filas de cuerpo y cada fila o columna puede tener 0 o mas celdas. Esta etiqueta consta de varios atributos y subetiquetas: La subetiqueta <tr> crea una fila dentro de la tabla y las subetiquetas <td><th> sirven para crear una celda, bien dentro de una fila, o de una cabecera.
Sus atributos principales son: rowspan que sirve para especificar el numero de filas que ocupa la celda y coldspan para especificar el numero de columnas de la celda.
La tabla a su vez, como la estructura del documento se subdivide en 3 partes:
<thead>, <tbody> y <tfoot>.

Mapa de imagenes.
Los mapas de imagenes sirven para hacer interactivo el documento que se visualiza en el navegador. Con la subetiqueta <area> permite especificar una region geometrica y el vinculo asociado a esa región. Además el atributo shape permite especificar el tipo de región (default,red,arcle,poli), con tcoords especificamos la posición en pantalla.
